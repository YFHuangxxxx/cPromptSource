## 数据集说明

详细说明见：https://hfl-rc.github.io/cmrc2018/

| 字段         | 描述内容     | 字段类型 |
| ------------ | ------------ | -------- |
| context_id   | 文本标识id   | string   |
| context_text | 文本内容     | string   |
| qas          | 问题与答案   | schema   |
| query_text   | 回答文本     | string   |
| query_id     | 回答标识符id | string   |
| answers      | 回答         | string   |
