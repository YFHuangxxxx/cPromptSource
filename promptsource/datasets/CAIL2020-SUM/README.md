## 数据来源

**CAIL 2020-司法人工智能挑战赛（司法摘要）**

本数据集使用的训练集、验证集、测试集来自于司法大数据院提供的法院裁判文书，约10000篇裁判文书以及对应的司法摘要，其中所涉及到的裁判文书均为民事一审判决书。

训练数据均包含若干行，每行数据为json格式，包含若干字段：

* ``id``：样本唯一标识符。
* ``summary``：样本的摘要内容。
* ``text``：将原裁判文书按照句子切分开，内含2个字段。
    * ``sentence``：表示句子的内容。
    * ``label``：表示句子的重要度。

实际测试数据不包含``summary``字段和``text``中的``label``。
